# Colorectal Cancer

This repository contains the Matlab implementation of a three-fault boolean network for Colorectal Cancer.

Step 1: Open the two files in Matlab.

Step 2: Set the number of combinations of drugs in line 1 of "script.m" (minimum:1, maximum:8, default is set to 3).

Step 3: Run "script.m".

The average size difference values will be computed and written to an excel file.
